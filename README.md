## 기능
- 로그인 및 회원가입 시 DB Connection 으로 유저 정보 Insert, Select
- Server, Client 소켓 통신
- 소켓 통신으로 그림판 실시간 동기화
- 게임 채팅창 다대다 채팅 구현
- 게임 시작 후 정해진 턴이 끝나면 게임 종료
